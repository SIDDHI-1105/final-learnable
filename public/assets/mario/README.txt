Place your Mario-style game assets in this folder. Use the following filenames:

- background.png      (background image)
- ground.png          (ground tile)
- mario-running.gif   (Mario running animation)
- mario-jumping.png   (Mario jumping sprite)
- mario-falling.png   (Mario falling sprite)
- block.png           (question/brick block)
- coin.gif            (animated coin)
- goomba.png          (enemy sprite)
- pipe.png            (green pipe)

You can use placeholder images for now. Replace them with real Mario sprites for a true Mario look.
